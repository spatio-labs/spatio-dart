# Spatio SDK

Generated SDK for the [SpatioAPI](https://www.spatio.app/developers/docs/api). Do not edit by hand — regenerated from the official OpenAPI spec on every release.

## Install

```bash
dart pub add spatio_sdk
```

## Import

```
import 'package:spatio_sdk/api.dart';
```

## Authentication

The SpatioAPI uses Personal Access Tokens. Mint one at **Settings → Tokens** in [Spatio Desktop](https://www.spatio.app/developers/docs/desktop) or on [spatio.app](https://www.spatio.app).
